Présentation

Le groupe Lacour est une holding qui se situe à Saint Doulchard (18230), elle comporte plusieurs interentreprises dirige par M. Angelo MAIDA. Le Groupe compte aujourd’hui environ 300 salariés dans tous les pôles. Ils proposent une gamme complète de produits et services informatiques pour les acteurs de l’après-vente automobile.
Le Groupe Lacour conçoit, développe, font la commercialisation, l’installation, la formation ainsi que l’assistance pour tous leurs logiciels.

Contexte



Je travaille sur le logiciel Fluide qui est une application web extranet pour le groupe A+ Glass. Fluide utilise un web service qui est programmé en C# et un IHM qui lui est programmé en ANGULAR, HTML et SCSS. Fluide s’appuie sur une base de données Mysql. A+ Glass doivent se vendre auprès des assureurs pour prouver qu’ils sont les meilleurs sur le marché et donc augmenter les prix. Il demande donc d’avoir sur Fluide une rubrique statistique qui renvoi un Excel avec différents statistiques.
